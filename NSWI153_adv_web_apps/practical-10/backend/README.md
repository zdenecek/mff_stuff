# NSWI153 Assignment - Slim Framework Application

This is a web application built with Slim Framework that demonstrates various web development concepts.

## Requirements

- PHP 7.4 or higher
- Composer

## Setup

1. Install dependencies:
```bash
composer install
```

2. Start the development server:
```bash
cd public
php -S 127.0.0.1:8090
```

3. Open your browser and navigate to:
```
http://127.0.0.1:8090
```

## Features

- Home page with application information
- Authors list 
- Author registration form
- Articles list
- Create article page 

## Project Structure

- `public/` - Public directory containing the entry point
- `src/` - Source code directory
- `templates/` - View templates
